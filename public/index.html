The Earth From Afar

This is a project written in Vanilla JavaScript with the use of THREE.js library. THREE.js is a powerful tool used to create and display animated 3D graphics in a web browser using WebGL. The possibilities you can develop are endless. If you'd like to learn more about the library, check out the GitHub Repo in the top right of the screen.
